A simple and efficient Student Record Management System developed as part of the Database Engineering (DBE) mini project.
This project demonstrates the use of basic SQL operations, including table creation, data insertion, updating, deletion, and retrieval of student information.
Add New Student Records
Update Existing Records
Delete Student Information
Display All Records
Search Students by ID/Name/Department
Use of SQL constraints such as PRIMARY KEY, UNIQUE, NOT NULL
Generate useful filtered reports using queries
